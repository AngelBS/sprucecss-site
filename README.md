## Contributing

1.  **Clone the repository**

    ```shell
    git clone https://github.com/conedevelopment/sprucecss-site.git
    ```

2.  **Install the dependencies**

    Navigate into your new siteâ€™s directory and install the dependencies.

    ```shell
    cd sprucecss-site/
    npm install
    ```

3.  **Start developing**

    To use Gatsby CLI first you have to install it globally by running: `npm install -g gatsby-cli`

    ```shell
    gatsby develop
    ```
