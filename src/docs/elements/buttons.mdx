## With Icon

To use any icon with Spruce, you must specify it in image format, preferably in SVG, because we set its height and width. Use with the ```btn__icon``` class combined with the alignment class: ```btn__icon-start``` (left) ```btn__icon-end``` (right).

<Preview>
  <div class="button-grid button-grid--stacked">
    <button type="button" class="btn btn--primary">
      <Icon type="arrow-right" className="btn__icon btn__icon--start" />
      Read more
    </button>
    <button type="button" class="btn btn--secondary">
      Read more
      <Icon type="arrow-right" className="btn__icon btn__icon--end" />
    </button>
  </div>
</Preview>

```html
<button type="button" class="btn btn--primary">
  <svg class="btn__icon btn__icon--start">...</svg>
  Read more
</button>
<button type="button" class="btn btn--secondary">
  Read more
  <svg class="btn__icon btn__icon--end">...</svg>
</button>
```

<Notification type="info">We use <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Logical_Properties">CSS Logical Properties</a> to position the icons hence the naming: start, end.</Notification>

## Configurable Variables

There are some defaults variables that you can configure to customize the button's look.

```scss
$btn-font-size:    $font-size-base;
$btn-font-size-sm: 0.8rem;
$btn-font-size-lg: 1.15rem;
$btn-font-style:   null;
$btn-font-weight:  500;
$btn-padding:      0.75em 1em;
$btn-padding-sm:   0.5em;
$btn-padding-lg:   0.9em 1.15em;
$btn-icon-size:    0.85em;
```

Also, Spruce manage the colors in a separate Sass map named $colors (under the btn object).

```scss
$colors: (
  btn: (
    primary-background: hsl(261, 52%, 59%),
    primary-background-hover: hsl(261, 52%, 40%),
    primary-foreground: hsl(0, 0%, 100%),
    secondary-background: hsl(350, 100%, 88%),
    secondary-background-hover: hsl(350, 100%, 92%),
    secondary-foreground: hsl(350, 100%, 20%)
  )
)
```

## Add Custom Button Variant

```scss
@mixin btn-variant(
  $color: primary-foreground,
  $background: primary-background
) {
  background-color: function.color(#{$background}, btn);
  color: function.color(#{$color}, 'btn');

  &:hover {
    background-color: function.color(#{$background}-hover, btn);
    color: function.color(#{$color}, btn);
  }

  &:focus {
    @include btn-focus($background);
  }
}
```
