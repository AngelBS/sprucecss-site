<p class="lead">Spruce comes with a bunch of Sass variables. It also uses CSS custom properties for colors and transitions.</p>

Our central concept is not to expose a variable that isnâ€™t needed for later customization, like the colors (for ```prefers-color-scheme```) or the transitions (for ```prefers-reduced-motion```).

[View Source on GitHub](https://github.com/conedevelopment/sprucecss/blob/main/scss/variable/_index.scss)

## Override Variables

There are two types of variables: single one and map. All of them are overwriteable, expandable with ```@use```'s ```with``` option.

```scss
@use '~sprucecss/scss/spruce' with (
  $colors: (
    base: (
      primary-darkest: hsl(272, 100%, 9%),
      heading: hsl(271deg, 100%, 9%)
    ),
    btn: (
      new-background: hsl(0, 0%, 95%), // Expanded new color values for buttons.
      new-background-hover: hsl(0, 0%, 90%),
      new-foreground: hsl(0, 0%, 10%),
    )
  ),
  $font-size-base: 1.063rem,
  $font-size-ratio: 1.250,
  $container-width: 86rem
);
```

You must do the overwrite at the very first use of ```@use```.

## CSS variables prefix

Spruce use the prefix for generating the custom properties.

```scss
$prefix: 'spruce';
```

Generated CSS custom properties:

```css
:root {
  --spruce-base-color-primary-darkest: #18002e;
  --spruce-base-color-heading: #18002e;
  --spruce-base-color-primary: #8660cd;
  --spruce-base-color-secondary: #ffc2cc;
}
```

[View Source on GitHub](https://github.com/conedevelopment/sprucecss/blob/main/scss/_root.scss)

## Settings

Control the main settings of Spruce through the ```$settings``` map.

You can access any setting with the [setting() helper function](/docs/sass/functions#setting).

```scss
$settings:(
  hyphens:  true, // Turn on/off the hyphenation
);
```

## Colors

We store the color values in a central map named ```$color```. It is a nested object separated by components.

To eliminate the repetition, we use some helper variables for the more generic colors (which we use more than once). You can overwrite these directly to make the theming easier or from the object one by one.

You can access any color with the [color() helper function](/docs/sass/functions#color).

```scss
$color-primary: hsl(261, 52%, 59%) !default;
$color-secondary: hsl(227, 92%, 55%) !default;

$color-black: hsl(205, 100%, 2%) !default;
$color-white: hsl(0, 0%, 100%) !default;

$color-gray-light: hsl(0, 0%, 90%) !default;
$color-gray: hsl(208, 7%, 30%) !default;

$colors: () !default;
$colors: map.deep-merge(
  (
    base: (
      primary: $color-primary,
      secondary: $color-secondary,
      background: $color-white,
      heading: $color-black,
      text: $color-gray,
      link: $color-primary,
      link-hover: color.scale($color-primary, $lightness: -20%),
      border: $color-gray-light,
      mark-background: hsl(50, 100%, 80%),
      mark-foreground: $color-black,
      code-background: hsl(261, 52%, 95%),
      code-foreground: $color-black,
      blockquote-border: $color-primary
    ),
    alert: (
      danger: hsl(2, 71%, 53%),
      info: hsl(202, 100%, 37%),
      success: hsl(160, 74%, 30%),
      warning: hsl(40, 96%, 32%)
    ),
    btn: (
      primary-background: $color-primary,
      primary-background-hover: color.scale($color-primary, $lightness: -20%),
      primary-foreground: $color-white,
      primary-foreground-hover: $color-white,
      secondary-background: $color-secondary,
      secondary-background-hover: color.scale($color-secondary, $lightness: -20%),
      secondary-foreground: $color-white,
      secondary-foreground-hover: $color-white
    ),
    form: (
      background: $color-white,
      background-disabled: hsl(0, 0%, 95%),
      border: $color-gray-light,
      border-disabled: $color-gray-light,
      border-focus: $color-primary,
      check-foreground: $color-white,
      check-background: $color-primary,
      label: $color-black,
      legend: $color-black,
      placeholder: hsl(208, 7%, 40%),
      text: $color-gray,
      select-foreground: $color-primary
    ),
    table: (
      border: $color-gray-light,
      hover: hsla(0, 0%, 0%, 0.05),
      stripe: hsla(0, 0%, 0%, 0.025)
    )
  ),
  $colors
);
```

## Spacers

You can access any spacer with the [spacer() helper function](/docs/sass/functions#spacer).

```scss
$spacer: 1rem;

$spacers: (
  xxs:  $spacer * 0.25,
  xs:   $spacer * 0.5,
  s:    $spacer,
  m:    $spacer * 1.5,
  l:    $spacer * 3,
  xl:   $spacer * 4.5,
  xxl:  $spacer * 5,
  xxxl: $spacer * 10
);
```

## Typography

You can access any font size with the [font-size() helper function](/docs/sass/functions#font-size).

```scss
$font-size-ratio: 1.333;

$font-size-base: 1rem;
$font-size-sm: 0.9rem;
$font-size-lg: 1.1rem;
$font-size-lead: math.pow($font-size-ratio, 1) * $font-size-base;

$font-sizes: ();
$font-sizes: map.merge(
  (
    h1: math.pow($font-size-ratio, 4) * $font-size-base,
    h2: math.pow($font-size-ratio, 3) * $font-size-base,
    h3: math.pow($font-size-ratio, 2) * $font-size-base,
    h4: math.pow($font-size-ratio, 1) * $font-size-base,
    h5: $font-size-base,
    h6: $font-size-base
  ),
  $font-sizes
);

$font-family-base: #{system-ui, -apple-system, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, 'Noto Sans', 'Liberation Sans', sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol', 'Noto Color Emoji'};
$font-family-heading: $font-family-base;
$font-family-cursive: #{SFMono-Regular, Menlo, Monaco, Consolas, 'Liberation Mono', 'Courier New', monospace};

$heading-font-weight: 700;

$line-height-sm: 1.2;
$line-height-md: 1.5;
$line-height-lg: 1.8;
$line-height-base: $line-height-lg;
$line-height-heading: calc(2px + 2ex + 2px);

$inline-text-element-padding: 0.1em 0.3em;
$inline-text-element-border-radius: $border-radius-sm;
```
